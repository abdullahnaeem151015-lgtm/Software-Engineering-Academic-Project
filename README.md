# Software Engineering Academic Project

This repository contains an academic Software Engineering assignment focused on core software modeling and design concepts. The project covers interaction, structural, and behavioral models used in software development.

## Topics Covered
- Interaction models and sequence diagrams  
- Structural models and class diagrams  
- Behavioral models and state diagrams  
- Layered and client-server architecture  
- Relationship between software design and implementation  

## Tools & Concepts
- StarUML (UML modeling)
- Software Engineering principles
- System analysis and documentation

## Purpose
The objective of this project is to demonstrate understanding of basic software modeling techniques and architectural concepts taught in undergraduate Computer Science coursework.

Academic Project
